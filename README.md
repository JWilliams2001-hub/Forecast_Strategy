# Fundamentals-Based Systematic Trading Strategies for the S&P 500 Energy Sector

Builds and back-tests two weekly systematic trading strategies for the S&P 500 Energy Sector Index (S5ENRS). Both strategies generate long/flat/short signals from a fair value gap — the spread between observed price and an econometrically estimated fair value — derived from macroeconomic inputs (1-year Treasury yield, S&P 500 P/E). Performance is evaluated strictly out-of-sample using a leakage-safe walk-forward design over Jan 2018 – Feb 2026.

---

## Research Objective

Can macro valuation fundamentals generate actionable, out-of-sample alpha in the U.S. energy sector at weekly frequency? This project tests two approaches to estimating time-varying fair value: rolling OLS regression and a Kalman filter.

---

## Methodology

| Component | Detail |
|-----------|--------|
| **Universe** | S&P 500 Energy Sector Index (S5ENRS) |
| **Frequency** | Weekly |
| **Signal inputs** | U.S. 1-year Treasury yield, S&P 500 trailing P/E, forward P/E |
| **Fair value model** | Log-linear regression: log(S5ENRS) ~ f(yield, P/E) |
| **Strategy 1** | Rolling OLS with short trailing window |
| **Strategy 2** | Kalman filter for time-varying coefficients |
| **Signal generation** | Long/flat/short on z-score of price deviation from fair value |
| **Back-test design** | Walk-forward; out-of-sample Jan 2018 – Feb 2026 |
| **Leakage prevention** | Signals at week *t* use only data through *t−1*; returns realized at *t+1* |

---

## Key Findings

| Strategy | Annualized Return | vs. S5ENRS Buy-and-Hold | vs. S&P 500 |
|----------|:-----------------:|:-----------------------:|:-----------:|
| Rolling OLS | **+7.97%** | +2.54 pp | −3.91 pp |
| Kalman Filter | −2.89% | Underperforms | Underperforms |
| S5ENRS Buy-and-Hold | 5.43% | — | −6.45 pp |
| S&P 500 | 11.88% | — | — |

The rolling OLS strategy delivers positive out-of-sample alpha relative to the energy sector but does not beat the broad market. The Kalman filter proves unviable at weekly frequency, likely due to excessive parameter uncertainty. Results suggest macro valuation variables carry incremental predictive content for energy returns, but excess returns over the broad market remain elusive.

Full analysis: [paper/forecasting_final_paper.pdf](paper/forecasting_final_paper.pdf)

## Running Locally

**Requirements:** Python 3.10+, Jupyter.

```bash
git clone <repo-url>
cd Forecast_Strategy
pip install -r requirements.txt
```

**Data setup:** The notebook reads from a `local_data/` directory that is not included in the repo. Create it and place the following files there:

```
local_data/
├── S5ENGS.csv                      # S&P 500 Energy Sector Index — weekly prices
├── energy_sector_master_data.xlsx  # S5ENRS prices, P/E, earnings yield (Bloomberg export)
└── sp500_treasury/
    ├── SP500 Price.csv             # S&P 500 Index — weekly prices
    ├── SP500 PE.csv                # S&P 500 trailing P/E
    ├── SP500 Best PE.csv           # S&P 500 forward P/E (consensus)
    └── H15T1Y.csv                  # U.S. 1-year Treasury yield (weekly)
```

Each CSV should have two columns: `Date` and `Last Price` (weekly frequency, 2010–2026).

**Public alternatives for the non-proprietary series:**
- 1-year Treasury yield: [FRED DGS1](https://fred.stlouisfed.org/series/DGS1)
- S&P 500 price: Yahoo Finance `^GSPC`

Once `local_data/` is populated, open the notebook:

```bash
jupyter notebook notebooks/fair_value_gap_strategy.ipynb
```

## Limitations

- The fair value model is parsimonious by design (2–3 inputs); richer factor sets may improve fit but risk overfitting at weekly frequency.
- Transaction costs and slippage are not modeled.
- The energy sector exhibits structural breaks (commodity cycles, ESG rotation) that a fixed rolling window may not fully capture.
- Out-of-sample window is approximately 8 years; results may not generalize across macro regimes.
